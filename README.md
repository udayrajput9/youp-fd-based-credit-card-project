# FD-Based Credit Card System 💳

A web-based FD-backed credit card system built using **Python Flask** and **SQLite**.  
This project simulates how banks issue credit cards against Fixed Deposits (FD) by assigning a credit limit based on the FD amount.

---

## 🚀 Features

- User Registration & Login
- Create Fixed Deposit (FD)
- Auto Credit Limit Calculation based on FD value
- View Credit Card Details
- Secure Backend using Flask
- Lightweight Database using SQLite
- Simple & Clean UI

---

## 🛠 Tech Stack

- **Backend:** Python (Flask)
- **Database:** SQLite
- **Frontend:** HTML, CSS, Bootstrap
- **Authentication:** Flask Sessions
- **ORM:** SQLite3 / SQLAlchemy (optional)
